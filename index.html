Introduction Of Courses

Web Technology Course!!

The Web Technology course is designed to prepare students for professional web design work. The class will be a mix of not only theoretical and soft skills, but also practical front-end and back-end techniques in web design. Upon completion of this course, students should have a thorough knowledge of all areas of web application design. Topics of front-end techniques include the knowledge of Android java programming, C# programming, HTML5 and CSS3, JavaScript, jQuery mobile and Bootstrap 4. In terms of back-end techniques, topics include building web servers, ASP.NET MVC scripting language, and MS SQL database. By the end of this course, students should have a solid understanding of the web design industry and modern APP design techniques
